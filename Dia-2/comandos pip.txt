pip (phyton install packages)

pip install NOMBRE_LIBRERIA         > instalar la libreria, si no lo encuentra lanzara un error
python -m pip install --upgrade pip > actualizar pip (gestor de paquetes) a la ultima version
pip freeze                          > lista todas las librerias instaladas en el scope actual
pip list                            > lista TODAS las librerias (instaladas manualmente y las instaladas predeterminadamente)
